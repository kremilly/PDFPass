## Example of use

* Example: ``pdfpass -f simple-file.pdf -p password -o simple-file-enc.pdf``

## Parameters

* *-f* or *--file*        Input file
* *-p* or *--password*    Password to encrypt
* *-o* or *--output*      Output file
